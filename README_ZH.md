## ✨ 核心特性

- **多数据库支持**: 内置数据库、SQLite、MySQL 等
- **数据模型**: 普通数据模型，自动同步表结构
- **丰富的数据类型**: 基础字段、选择器、附件、组织架构、关联数据等 30+ 种类型
- **查询构建器**: 灵活的数据查询接口
